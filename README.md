<h3>Penjelasan</h3>
Aplikasi Go-Banten adalah suatu aplikasi berbasis web yang memfasilitasi para pembeli dan penjual banten. Pada aplikasi ini melibatkan 3 jenis pengguna yaitu administrator, penjual dan pembeli.
